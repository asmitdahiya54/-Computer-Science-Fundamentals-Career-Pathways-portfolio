# Development Process

- All projects were developed using Visual Studio Code.
- Git was used to version control the projects.
- GitHub was used to host the repository.
- Steps:
  1. Created project folder and files in VS Code.
  2. Added images and assets.
  3. Wrote HTML, CSS, and JavaScript code.
  4. Tested each project in a browser.
  5. Committed changes to GitHub regularly.
